# moljs

A web-based 3D molecule viewer built with React, TypeScript, and [3Dmol.js](https://3dmol.csb.pitt.edu/).

## Live Demo

[**View the hosted application here**](https://tealwp.github.io/moljs)

## Usage

This application allows you to visualize molecular structures directly in your browser.

### Controls

- **Rotate**: Click and drag with the left mouse button.
- **Zoom**: Use the mouse scroll wheel.
- **Pan**: Click and drag with the right mouse button (or Ctrl + Left Click).

## Development

### Installation

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```

### Run Locally

```bash
npm run dev
```

### Build & Deploy

To build for production and deploy to GitHub Pages:

```bash
npm run deploy
```
